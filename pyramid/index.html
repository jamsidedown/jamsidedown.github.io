Learn by doing puzzle 3

Pyramid paths

Find the maximum sum from adding the nodes in a path from the top of each hexadecimal pyramid to the bottom, adding the node down to the left or down to the right each step to finish anywhere on the bottom row.

Example 1

5
3 4
2 1 7
c d 2 5

Here the optimal path, shown in red, gives us the maximum sum of 23:

5 + 4 + 1 + d = 0x17 = 23

The input file for this example is available here: example1.txt

Example 2

With 3D pyramids, each node in the path has 4 options for the next node to navigate to; back left, back right, front left, and front right.
